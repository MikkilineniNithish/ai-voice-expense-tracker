# 💰 AI Voice Expense Tracker

AI Voice Expense Tracker is a smart finance management web application that allows users to track daily expenses using both manual entry and voice commands.

The application provides real-time expense tracking, category management, monthly budgeting insights, and an interactive dashboard for better personal finance management.
